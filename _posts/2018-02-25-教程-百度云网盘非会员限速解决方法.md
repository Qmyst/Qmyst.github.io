# 使用idm

<p>使用 <a href="http://hanayuki.me/img/IDMan628Build5-Yanu.exe" style="color:red">IDM下载工具（点我）</a></p>

# 安装浏览器插件

<p>安装 <a href="http://tampermonkey.net/" style="color:red">tampermonkey（点我）</a></p>

# 安装百度网盘直链插件

<p>安装 <a href="https://greasyfork.org/zh-CN/scripts/25729-%E7%99%BE%E5%BA%A6%E7%BD%91%E7%9B%98%E7%9B%B4%E9%93%BE" style="color:red">百度网盘直链（点我）</a></p>

# 开始下载

<p>在自己的网盘上新建一个文件夹，再在这个文件夹里新建多个文件夹（五个左右即可）</p>
<p>把想下载的文件保存到新建文件夹中（和其他多个文件夹并列在同一个文件夹），如下图</p>
<img src="http://hanayuki.me/img/post-baidu01.jpg" style="width:100%">
<p>选中最外层的新建文件夹点击下载，这时会弹出下载窗口（各浏览器有所差异），复制下载链接，打开idm，添加任务，粘贴刚复制的链接开始任务即可</p>
